OPENPROCESSING.COM

Halo of Gradient (P5js.)

Time: Sep. 2021
Course: Creative Technology 1
Instructor: Maxim Safioulline

Brief
Use randomness and/or Perlin noise to create organic shapes. Add controls for the user to interact and modify them.

DESIGN PORCESS

This P5js project simulate the light ring. I used random() for color details, noise() for traces and createSlider() to make color adjustable from green to yellow. 

At first I did’t notice that we are asked to make organic shapes! So I tried geometry. I was studying noise at this time.

Added a ellipse. I want to make it spin and bump back when at edge, but I didn’t know how to use if() to vel * -1 when the grid is also spinning so I gave up.

Studying how to write arguments in if().

I kept on adjusting the values in for() to make the light ring the most pleasant way, making it looks “organic”.

When I add a slider I try several time on how to use the variable, wondering where to place the code.

I tried position but that’s wired, so finally I used slider to control color of the inside small one.
