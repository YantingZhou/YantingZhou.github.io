CSS webpage

Time: Jan. 2022
Course: Creative Technology 2
Instructor: Maxim Safioulline
Tool: HTML / CSS, Github

Project Brief:

Create 3 HTML pages. Each page should contain a link menu to enable navigation among them, with the live links pointing to other pages. Each of the three pages should contain a unique layout using CSS grid technique.

DESIGN PROCESS

No.1 Text & Image

  • I watch this video (left) to make my first page (right). I learnt fr in grid means fraction, similar to flexbox.

No.2 Stacking Images

  • In my second page I want all be photos, but the alignment can be more fun, I think they can overlap on each other.
  • I copy the photo, then realizing a device-width page could be more of aethetics for pictures. But there is a new problem, that the picture size is fixed. If using a larger monitor, there would be gaps between each other. I googled for how to let picture size in grid cell adjustable with device window like a flex-box.
  • Then I found this code on right and later applied into my file. Now I have more flexible pictures.
  • Then I arrange the photos. Making them across the grid cells.
  • First I tried to copy the .grid-col-span-2 and adapt it into.grid-row-span-2, but I found it didn’t work when I add both of these into class of <img>, I don’t know if it can only have two class name or it simply doesn’t work when applying both col and row. 
  • I notice this video used another syntax :nth-child, I found it on W3School and figured out its function is just what I need - make pics occupy both column and row. 
  • Now I have this. The first picture did occupy 2*2 cells, but its size won’t change, that’s another challange.
  • Soon I found the error of code. It was the .gridItem>img in CSS that went wrong. In my HTML code, class ”gridItem” is inside the <img>, so I just delete the .gridItem in CSS, and things went right.
  • width: 100% functioned. Similarly I added height: 100% to format the height.
  • In the end, I use inspect window to adjust each cells to make a photo stacking effect.

No.3 Bad Cat

  • I created this page with the mindset of having fun and trying things out. Exploring different features of CSS, this page ended up as a web collage.
