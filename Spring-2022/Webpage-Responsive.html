Responsive webpage

Time: Feb. 2022
Course: Creative Technology 2
Instructor: Maxim Safioulline
Tool: HTML / CSS, Github

Project Brief:

Create a page that has the same content, but completely changes the way it's presented depending on the devive you use to view it. Colors, layout, typography, composition - all needs to exist in at least three versions, but on the same page.

DESIGN PROCESS

Fit image into grid cell

  • I want to make image in grid fit grid height. I first try to add max-height:100% in css grid item (inside #gi1, grid.css), but it did not work because thatâ€™s the height of grid cell rather than the image inside. Then I wrote code on the left in css.
  • but it is the cell becoming larger, fitting to the image size, rather than image fit in grid size. 
  • Then I googled for solution, I found it is because the size of gird only have minimum
  • but the picture is distorted, so I added object-fit: cover in img in style.css.

Responsive Circles

  • I draw some dots in one of the cell, I want them to be responsive in number: Show 2 on smallest screen, 4 in medium screen, and 9 on large screen. I was confused, since we learnt that responsive webpage is written from small to large, then how can I add more things in big screen?
  • I suppose, since the html code is not going to be changed, so the all 9 dots must pre-exist.
  • I search for how to hide element in div, and found two syntax: visibility and display. They can both use in hiding things, but: visibility has two opposite expression, visibility: visible and visibility: hidden; display only have one, display: none. 
  • So, for controling if an element is hidden or not, visibility is more suitable.
.dot {
    height: 180px;
    width: 180px;
    background-color: hotpink;
    border-radius: 50%;
    display: inline-block;
}
.d2 {
    visibility: hidden;
}
.d3 {
    visibility: hidden;
}
/*MEDIUM SCREEN*/
@media (min-width: 768px){
    .dot {
        height: 140px;
        width: 140px;
    }
    .d2 {
        visibility: visible;
    }
}
/*LARGE SCREEN*/
@media (min-width: 960px) {
    .dot {
        height: 100px;
        width: 100px;
    }
    .d3 {
        visibility: visible;
    }
}
